### Pasos para poder usar PPPMaker:

1.- Baja el archivo en formato  Zip o clona mediante un `git clone https://github.com/cesaroer/PPPMaker.git`
2.- Una vez tengas el archivo en tu PC posicionate en la carpeta donde lo hayas bajado.
3.- Entra a la carpeta PPPMaker y abre el archivo PPPMaker.xcodeproj
4.- Ya en Xcode Ve a la parte superior y da click en el simbolo de **play**
5.- Espera a que cargue el simulador y que termine de compilar PPPMaker
